# 贡献指南

感谢你对 SkillWisp 的关注！

## 贡献 Skills

1. Fork 本仓库
2. 在 `skills/` 目录下创建你的 Skill
3. 提交 Pull Request

### Skill 格式

```
skills/
└── @your-name/
    └── your-skill/
        └── SKILL.md
```

SKILL.md 格式：

```markdown
---
name: your-skill
description: 一句话描述
author: @yourname
version: 1.0.0
tags: [tag1, tag2]
---

# Skill 内容

详细说明...
```
